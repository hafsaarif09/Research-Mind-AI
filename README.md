## 🌟 What is ResearchMind AI?
ResearchMind AI helps students and researchers discover, analyze, and understand any research topic in minutes using AI — instead of spending hours reading papers manually.

## ✨ Features
| Feature | Description |
|---------|-------------|
| 📊 Smart Summary | Instant AI-generated research overview |
| 🎯 Gap Finder | Discover unexplored research areas |
| 📈 Trend Analyzer | Visualize publication growth over time |
| 🧠 Mind Map | Interactive knowledge structure |
| 💾 Dataset Explorer | Find relevant datasets instantly |
| 📚 Literature Review | Auto-generated academic writing |
| 🤖 AI Chatbot | Real-time research Q&A (Groq API) |
| 📖 Research Library | Save and organize your research |

## 🛠️ Tech Stack
- **Frontend:** React + Tailwind CSS + Framer Motion
- **Backend:** FastAPI + Python
- **AI:** Groq API (Mixtral-8x7b) + Google Gemini
- **Charts:** Recharts

## 🚀 Quick Start

### Frontend
```bash
npm install
npm start
```
Opens on `http://localhost:3000`

### Backend
```bash
cd backend
python -m venv venv
venv\Scripts\activate
pip install -r requirements.txt
python -m uvicorn main:app --reload --port 8000
```
Runs on `http://localhost:8000`
